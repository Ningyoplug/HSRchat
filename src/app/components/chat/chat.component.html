How to use images

There are two ways to make a character send an image: by using an image URL, meaning that image needs to already exist on the internet, or by uploading an image from your device.

The image URL method is the more reliable one.

If you use a URL, there should be no issues when you decide to save your conversation via the save feature.
However, if you upload an image from your device, even if you save the the conversation, the image will disappear once you refresh the page.

How to get an image URL:

Upload the image somewhere on the internet first. You can post it on social media (eg. Twitter) or use a site like ImgBB.

Once your image has been posted online, right click it and select "Copy image URL".
You can now paste the URL that links to your image into the chat box!
